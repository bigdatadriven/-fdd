Opening

Welcoming everyone

09:00 AM US-EDT

Débuter avec Azure pour les DBA - Philippe Geiger

Description:
Pendant une heure, nous allons détailler les fonctionnalités que tout DBA doit connaître à propos d'Azure et de ses fonctionnalités orientés Data.

10:00 AM EDT-US

IaC another way for automating SQL Server provisioning - David Barbarin

Description:
Automating SQL Server installation is not something new and PowerShell was a good driver for a while. But the raise of IaC provides new ways and paradigms for infrastructure including declarative langages like Ansible or Terraform and consistent and repeatable deployments. Does it fit well with SQL Server? Join to this session to get the answer.

11:00 AM EDT-US

Groupes de disponibilité : configuration pas à pas avec PowerShell et dbaTools - Christophe Laporte

Description:
De la création d'un cluster Windows jusqu'à la configuration d'un groupe de disponibilité, cette session didactique vous accompagne sur la mise en oeuvre complète via des scripts PowerShell et dbaTool

12:00 - 12:25 PM EDT-US

BREAK

12:30 PM EDT-US

Set up your data and Azure Synapse in minutes and start analyzing your data immediately! - Michelle Gutzait

Description:
In this session, Michelle will show you how to push your existing data from your relational databases and files to the Azure Data Lake storage, set up Azure Synapse Analytics workspace and start exploring your data immediately!

01:30 PM EDT-US

Power BI Premium, la mort de Azure Analysis Services? - Charles-Henri Sauget & Marie Aubert

Description:
Power BI Premium est sorti en 2017 et a connu un développement massif de la part de Microsoft depuis.
Les dernières innovations majeures sont le Gen2, et le Premium Per User (PPU) qui sont sortis cette année.
Ces évolutions montrent la volonté de Microsoft d'imposer le Premium dans le monde des solutions industrialisées. De plus, au fil du temps, l'écart de fonctionnalités entre Premium et Azure Analysis Services s'élargit. Premium est désormais présenté par Microsoft comme un SuperSet AAS, mais qu'est-ce que cela signifie ?
- Est-il déjà possible de migrer d'AAS vers Premium ?
- Si oui, avec quels efforts et contraintes ?
- Quelles sont les différences de performances ?
- Quelle est la différence de prix ?
- En quoi l'arrivée de Power BI Premium Gen2 a changé la donne ?
Venez obtenir des retours du monde réel.

02:30 PM EDT-US

How Azure Purview can help your data platform? - Charles-Henri Sauget & Matthieu Roy

Description:
Le Data Catalog est un sujet sensible et controversé de l'entreprise.
Par sa nature transverse, il phagocyte assez vite de perpétuelles frustrations du métier qui aimerait savoir :
- où sont ses données ?
- comment sont-elles transformées ?
- comment sont-elles liées entre elles ?
- qui sont les contacts ?

Et les équipes projets ainsi que les développeurs qui :
- sont réfractaires à la documentation
- ne voient pas toujours l’intérêt d'exposer les données
- ont toutes une approche différente de l'implémentation d'un projet de données

Un des sujets compliqué techniquement et politiquement à l'image d'un projet de Master Data Management (MDM), qui lie organisationnel et système d'information. Avec qui plus est, un retour sur investissement (ROI) assez difficile à calculer.

Microsoft propose depuis fin 2021 une solution technique basée sur Atlas, Azure Purview. Nous réunirons autour de la table 3 profils complémentaires afin de discuter Data Catalog et implémentation Purview.

03:30 PM

Power BI : les bonnes pratiques - Philippe Geiger

Description:
Comme beaucoup de produits, Power BI offre ses meilleurs résultats si nous respectons certaines règles.
Pendant cette session, nous détaillerons les bonnes pratiques à connaître et à appliquer afin que les performances de nos rapports soient les plus optimales. La session sera axée sur de nombreuses démonstrations et laissera aussi la place à des échanges de questions/réponses.

04:30 PM EDT-US

Data quality with Azure Synapse Analytics and Data Factory - Christian Cote

Description:
Data quality is a topic that is present in every data projects. How can we validate our data, why data profiling is so important? This session will talk about these topics and demo how to achieve data quality pipelines with both Azure Synapse Analytics and Data Factory.

05:30 PM EDT-US

Connect Power BI with SQLite and Primavera P6 - Vahid Doustimajd

Description:
Primavera P6 is the most potent, practical and one of the main software that experts use in Project Management for scheduling, and in that field, reports and dashboards are critical. The P6 Standalone has an SQLite database, and Power BI does not have a Certified Connector for the SQLite, so in this meeting, I'll explain how to connect Power BI with SQLite and Primavera to create a report.
