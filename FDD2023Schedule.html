The links to the sessions will be available 8 hours before the live event. In order to join a session, (1) Scroll down to find the session that you wish to attend. (2) Find the room that the session is in. and (3) click the link of the room to join - this will open Teams windows. Note that you can join using Teams desktop app or using the browser. If you have any issue then try to use incognito mode (private mode) of the browser/
